# 개발일지 — 2026-06-08 테마 전환 기능 추가

## 작업 목표
상단 메뉴 우측 끝(상담 신청 버튼 뒤)에
1) 다크/라이트 모드 전환 버튼
2) 5가지 컬러로 바꿀 수 있는 컬러 팔레트 선택기
를 배치.

## 작업 내용

### 1. 테마 변수 구조 확장 (`src/index.css`)
- `:root` = 라이트 모드 기본값
- `:root[data-mode='dark']` = 다크 모드 중립색 오버라이드
  - surface/surface-2/ink/text/muted/line/nav-bg/shadow 재정의
- `:root[data-palette='...']` = 주컬러(navy/royal) 5종 오버라이드
  - **royal**(로열블루·기본) / **emerald**(에메랄드) / **crimson**(크림슨) / **violet**(바이올렛) / **amber**(앰버)
- `--nav-bg` 변수 신설로 네비게이션 배경도 모드 대응

### 2. 하드코딩 색상 → 변수화 (`src/App.css`)
- 네비/카드/타임라인 점/모바일 메뉴/폼 입력의 `#fff` → `var(--surface)`
- 브랜드·네비 텍스트·햄버거 `var(--navy)` → `var(--ink)` (다크모드 가독성)

### 3. 상태 관리 훅 (`src/theme/useTheme.js`)
- mode/palette 상태를 `localStorage`에 저장·복원
- `document.documentElement` 에 `data-mode`, `data-palette` 속성 반영
- `PALETTES` 5종 정의(라벨·대표색)

### 4. 전환 UI (`src/components/ThemeSwitcher.jsx`)
- 🌙/☀️ 토글 버튼 + 🎨 팔레트 드롭다운(바깥 클릭 시 닫힘)
- 선택 중인 팔레트에 체크 표시

### 5. 네비게이션 배치 (`src/components/Navbar.jsx`)
- `.nav-right` 그룹으로 재구성 → 상담 신청 버튼 **뒤**에 스위처 배치
- 데스크톱: 메뉴 우측 끝 / 모바일: 상단 바에 항상 노출(햄버거 옆)

### 6. FOUC 방지 (`index.html`)
- 페인트 이전 인라인 스크립트로 저장된 모드/팔레트 즉시 적용
