# 개발일지 — 2026-06-08 CTA 버튼 가독성 개선

## 문제
상단 메뉴 "상담 신청" 버튼(`.btn-primary`)이 `var(--royal)` 배경 + 흰 글자를 사용.
컬러 팔레트 전환 시 일부 색에서 흰 글자 대비가 부족(가독성 결여).

대비 측정(흰 글자 기준):
- emerald `#059669` ≈ 2.9:1 ✗ (WCAG AA 미달)
- amber `#d97706` ≈ 2.4:1 ✗
- crimson `#dc2626` ≈ 4.0:1 (보통 텍스트 경계)

## 해결
팔레트마다 **흰 글자 AA를 통과하는 전용 CTA 색** 변수 도입.
- `--cta-bg` / `--cta-bg-hover` 신설 (`src/index.css`)
  - royal `#1d4ed8` / emerald `#047857` / crimson `#b91c1c` / violet `#6d28d9` / amber `#b45309`
- `.btn-primary` 가 `var(--royal)` 대신 `var(--cta-bg)` 사용하도록 변경
  - 호버·그림자도 동일 계열로 통일

## 효과
- 5개 팔레트 + 다크/라이트 전 조합에서 버튼 글자 대비 확보
- 상담 신청 외 모든 기본(primary) 버튼에 일괄 적용

## 결과
`npm run build` 성공, 커밋·푸시·gh-pages 배포 완료.
