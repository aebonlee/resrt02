# 개발일지 — 2026-06-08 이모지 → Material Symbols 교체

## 목표
사이트 전체의 컬러 이모지를 Google Material Symbols(모노크롬 아이콘 폰트)로 교체.
인위적인 이모지 느낌을 줄이고 감성적이고 정제된 디자인으로 통일.

## 작업 내용

### 1. 아이콘 폰트 도입
- `index.html` 에 Material Symbols Rounded 웹폰트 `<link>` 추가 (preconnect 포함)
- `src/index.css` 에 `.material-symbols-rounded` 기본 스타일
  - `font-variation-settings`: 가는 두께(wght 300)로 감성적 톤, `.fill` 변형 추가
- 재사용 컴포넌트 `src/components/Icon.jsx` 신설 (`<Icon name="..." fill />`)

### 2. 이모지 → 아이콘 매핑
| 위치 | 기존 | 변경(Material) |
|------|------|----------------|
| 프로그램 개발 | ⚙️ | `terminal` |
| 웹 개발 | 🌐 | `language` |
| 컴퓨터 판매 | 💻 | `computer` |
| 유지보수 | 🛡️ | `shield` |
| 고객 중심 | 🎯 | `favorite` |
| 기술력 | 🔧 | `build` |
| 신뢰 | 🤝 | `handshake` |
| 이메일/전화/주소/시간 | ✉️📞📍🕒 | `mail`/`call`/`location_on`/`schedule` |
| 다크·라이트·팔레트 | 🌙☀️🎨 | `dark_mode`/`light_mode`/`palette` |
| 메뉴·닫기 | ☰✕ | `menu`/`close` |
| 히어로 배지 | 🚀 | `auto_awesome` |
| 체크 표시 | ✓ | `check` / `check_circle` |
| 포트폴리오 썸네일 | 🌐🖥️⚙️ | `language`/`dns`/`terminal` |

### 3. 적용 범위
- `company.js`(서비스), Home/About/Services/Portfolio/Contact, Navbar, ThemeSwitcher
- `svc-points` 체크는 CSS `::before` 에 Material 리거처(`content: 'check'`)로 처리
- `form-note` flex 정렬로 아이콘+텍스트 정렬

### 4. 비고
- 버튼의 화살표 `→` 는 이모지가 아닌 타이포그래픽 기호로 유지
- 아이콘은 `currentColor` 상속 → 다크모드/팔레트 색상과 자연스럽게 연동

## 결과
`npm run build` 성공, dev/폰트 엔드포인트 HTTP 200 확인. 커밋·푸시·gh-pages 배포 완료.

- Font Awesome 6.7.2 CDN 로드(index.html), Material/FA 두 폰트 동시 로드로 즉시 비교 가능
